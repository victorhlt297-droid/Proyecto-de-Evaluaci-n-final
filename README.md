*Descripción del Proyecto

Este proyecto consiste en el desarrollo de un programa en Python donde la computadora intenta adivinar el número pensado por el usuario entre 1 y 100.

El sistema utiliza un algoritmo eficiente basado en búsqueda binaria para reducir el rango de posibilidades en cada intento.

*Objetivo del Programa

Aplicar los conocimientos adquiridos en las cuatro unidades de la materia mediante el desarrollo de un software funcional que utilice:
-Variables
-Estructuras condicionales
-Ciclos repetitivos
-Programación estructurada
-Programación funcional
-Optimización de algoritmos

*Principales Funcionalidades

- Solicita al usuario pensar en un número entre 1 y 100
- Calcula automáticamente el número intermedio del rango
- Ajusta el rango según la respuesta del usuario
- Cuenta la cantidad de intentos
- Finaliza cuando el número es adivinado
- Maneja errores en caso de respuestas inválidas

*Algoritmo Utilizado

El programa implementa el algoritmo de búsqueda binaria, el cual divide el rango en cada intento, reduciendo significativamente la cantidad de pasos necesarios para encontrar el número correcto.

Este método demuestra eficiencia y optimización en la toma de decisiones.
